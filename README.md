Калькулятор финансовых целей
Описание:
	Десктопное приложение на платформе WPF (C#), предназначенное для планирования и отслеживания финансовых целей пользователя. Приложение позволяет создавать цели, управлять накоплениями, отслеживать прогресс достижения и сохранять данные в базе данных PostgreSQL.

Пример использования: 
	накопление на покупку
	путешествие 
	сбережения	
	инвестиции

Функциональные возможности:
	Список финансовых целей
		Отображение всех целей пользователя с указанием:
			названия
			категории
			приоритета
			дедлайна
			текущего статуса
			прогресса накоплений (текущая сумма / целевая сумма)
			
	Добавление финансовой цели
		При создании цели пользователь указывает:
			название
			описание
			целевую сумму
			начальную сумму
			дедлайн
			категорию
			приоритет
			план накоплений (частота и сумма)
	Редактирование цели
		В карточке цели доступно редактирование:
			названия
			описания
			целевой суммы
			текущей суммы
			дедлайна
		Изменения сохраняются в базе данных. Статус цели пересчитывается автоматически.

	Пополнение цели
		Пользователь может вручную добавлять взносы.
		При достижении или превышении целевой суммы статус цели автоматически меняется на «Достигнута».

	Сортировка и фильтрация
		Реализована:
			фильтрация по статусу
			фильтрация по категории
			фильтрация целей с близким дедлайном
			сортировка по дате создания
			сортировка по дедлайну
			сортировка по приоритету
			сортировка по прогрессу достижения
		Визуальное отображение прогресса
		Для каждой цели используется ProgressBar, наглядно показывающий процент накоплений.

	Экспорт данных
		Данные целей можно экспортировать в файл JSON с выбором пути сохранения.
		Служебные и чувствительные данные (пользователь, пароли) в файл не сохраняются.

	Авторизация пользователей
		Поддерживается регистрация и вход пользователей.
		Каждый пользователь видит только свои цели.

Архитектура и технологии
	Язык программирования: C# 14.0
	Платформа: WPF
	База данных: PostgreSQL
	ORM: Entity Framework Core
	Используется Dependency Injection
	Архитектура основана на принципах ООП:
	инкапсуляция (вся бизнес-логика вынесена в сервисы)
	использование интерфейсов
	разделение UI и логики работы с данными

Как работать с приложением
	Запустить приложение
	Зарегистрироваться или войти под существующей учетной записью
	Создать финансовую цель

Управлять целями:
	редактировать параметры
	пополнять сумму
	отслеживать прогресс
	
При необходимости экспортировать данные в JSON

Настройка базы данных
	Приложение использует PostgreSQL.

Настройка строки подключения
	Открыть файл appsettings.json, расположенный рядом с App.xaml, и указать строку подключения:
		{
			"ConnectionString": "Host=localhost;Port=5432;Database=FinancialGoals;Username=postgres;Password=your_password"
		}

Установка зависимостей
	Необходимо установить:
		PostgreSQL
		.NET SDK соответствующей версии
		
Создание базы данных (миграции)
	В папке проекта выполнить команды:
		Добавление миграции:
		dotnet ef migrations add InitialCreate

		Применение миграции:
		dotnet ef database update

		Если база данных уже существует и требуется пересоздание схемы:
		dotnet ef database drop
		dotnet ef database update

		Внимание: команда drop удаляет все данные.

Примечания
	Строка подключения не хранится в коде
	Вся работа с базой данных выполняется через сервисный слой
	Проект готов к расширению (например, импорт JSON, аналитика, графики)
